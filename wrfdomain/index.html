Acerca de

El Asistente de Dominios WRF es una aplicación de una sola página para simplificar la creación, visualización y modificación del archivo de entrada namelist.wps del WRF Preprocessing System (WPS) namelist.wps input file.

Archivos Geográficos

Utilice esta pestaña para mostrar características geográficas de varios archivos de datos. Esto está destinado a ayudar en el diseño de dominios WRF basados en las ubicaciones de las características mostradas.

La herramienta actualmente admite los formatos de archivo GeoJSON, KMZ, KML, GPX y WPT. Si se necesita un formato de archivo adicional, por favor, envíe una solicitud de función en la página de problemas del proyecto en GitHub.
